01

Tentang PialaSport

PialaSport adalah sistem otomasi konten sepakbola yang mengumpulkan berita olahraga dari sumber terpercaya, lalu secara otomatis memposting konten tersebut ke berbagai platform media sosial.

Kebijakan Privasi ini menjelaskan bagaimana kami menangani data dalam proses operasional bot ini.

02

Platform yang Diakses

PialaSport terhubung ke platform berikut untuk keperluan posting konten otomatis:

  • 📘 Facebook
  • 📸 Instagram
  • 🎵 TikTok
  • ✈️ Telegram

Akses ke platform tersebut dilakukan menggunakan token akses resmi yang diberikan oleh masing-masing platform melalui program developer mereka (Meta Graph API, TikTok for Developers, Telegram Bot API).

Bot ini tidak membaca pesan pribadi, tidak mengakses kontak pengguna lain, dan tidak memiliki akses ke data akun selain izin yang diperlukan untuk memposting konten ke halaman/channel yang dikelola.

03

Data yang Dikumpulkan

PialaSport tidak mengumpulkan data pribadi pengguna. Data yang diproses oleh sistem ini terbatas pada:

  • Konten berita olahraga — diambil dari RSS feed dan API berita publik (Detik, Bola.net, Goal.com, dll)
  • Data statistik sepakbola — dari API Football untuk informasi pemain dan pertandingan
  • Gambar publik — background dari Unsplash API menggunakan kueri tematik
  • Log operasional — catatan internal tentang jadwal posting, status pengiriman, dan error sistem (disimpan lokal, tidak dibagikan)

Semua data yang diproses bersifat konten publik terkait olahraga dan digunakan semata-mata untuk tujuan pembuatan dan distribusi konten sepakbola.

04

Penggunaan API Pihak Ketiga

Sistem ini menggunakan API dari layanan berikut:

  • Anthropic Claude API — untuk generate teks konten secara otomatis
  • Football API (api-sports.io) — data pemain dan statistik
  • Unsplash API — gambar latar bertema olahraga
  • Meta Graph API — posting ke Facebook Page dan Instagram Business
  • TikTok for Developers API — posting video/konten ke akun TikTok
  • Telegram Bot API — posting ke channel Telegram

Setiap layanan pihak ketiga memiliki kebijakan privasi tersendiri. Kami menyarankan Anda membaca kebijakan privasi masing-masing platform.

05

Penyimpanan & Keamanan Data

Token akses API disimpan dalam file konfigurasi lokal (.env) yang tidak pernah diunggah ke repositori publik atau layanan pihak ketiga.

Log operasional disimpan secara lokal di server yang dikelola oleh operator dan tidak dibagikan kepada pihak mana pun.

Sistem ini tidak menggunakan database untuk menyimpan data pengguna.

06

Hak Pengguna

Karena PialaSport tidak mengumpulkan data pribadi dari pengguna, tidak ada profil pengguna yang perlu dihapus atau diubah.

Jika Anda menemukan konten yang bermasalah atau ingin mengajukan pertanyaan terkait privasi, silakan hubungi kami melalui informasi kontak di bawah ini.

07

Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu. Perubahan signifikan akan diumumkan melalui channel Telegram resmi PialaSport. Tanggal pembaruan terakhir selalu dicantumkan di bagian atas halaman ini.

01

About PialaSport

PialaSport is a football content automation system that collects sports news from trusted sources and automatically publishes content to various social media platforms.

This Privacy Policy explains how data is handled during the operational process of this bot.

02

Platforms Accessed

PialaSport connects to the following platforms for the purpose of automated content posting:

  • 📘 Facebook
  • 📸 Instagram
  • 🎵 TikTok
  • ✈️ Telegram

Access to these platforms is performed using official access tokens granted by each platform through their developer programs (Meta Graph API, TikTok for Developers, Telegram Bot API).

This bot does not read private messages, does not access other users' contacts, and has no access to account data beyond the permissions required to post content to managed pages/channels.

03

Data Collected

PialaSport does not collect personal user data. Data processed by this system is limited to:

  • Sports news content — fetched from RSS feeds and public news APIs (Detik, Bola.net, Goal.com, etc.)
  • Football statistics data — from Football API for player and match information
  • Public imagery — background images from Unsplash API using thematic queries
  • Operational logs — internal records of posting schedules, delivery status, and system errors (stored locally, never shared)

All processed data consists of publicly available sports-related content and is used solely for the creation and distribution of football content.

04

Third-Party API Usage

This system uses APIs from the following services:

  • Anthropic Claude API — for automated text content generation
  • Football API (api-sports.io) — player data and statistics
  • Unsplash API — sports-themed background images
  • Meta Graph API — posting to Facebook Page and Instagram Business
  • TikTok for Developers API — posting video/content to TikTok account
  • Telegram Bot API — posting to Telegram channel

Each third-party service has its own privacy policy. We encourage you to review the privacy policy of each respective platform.

05

Data Storage & Security

API access tokens are stored in a local configuration file (.env) that is never uploaded to public repositories or third-party services.

Operational logs are stored locally on the operator-managed server and are not shared with any party.

This system does not use a database to store user data.

06

User Rights

Since PialaSport does not collect personal data from users, there are no user profiles to delete or modify.

If you encounter problematic content or wish to raise a privacy-related inquiry, please contact us via the contact information below.

07

Policy Changes

This Privacy Policy may be updated at any time. Significant changes will be announced through the official PialaSport Telegram channel. The last updated date is always displayed at the top of this page.
